Drawing App

Description

This assignment was to create a drawing app using event blocks. To draw a polygon, press space and the app will tell you the largest polygon you can draw from the sprites current position. To draw a circle, press X answer the 4 questions. To draw a squiral with a custom angle, press F. To set the color for a polygon, circle, or squiral, press the number of the color before pressing space, X, or F.

Controls:

• Arrows: Move

• Cilck and Drag: Free draw

• Space: Polygon drawer

• F: Squiral

• X: Circle drawer

• C: Clear

• S: Stamp

• M: Go to 0,0

• D: Pen Down

• U: Pen Up

• Q: Decrease pen size

• W: Increase pen size

• E: Reset pen size

• L: Preset drawing

• 0: Eraser

• 1: Red

• 2: Orange

• 3: Yellow

• 4: Green

• 5: Blue

• 6: Purple

• 7: Pink

• 8: Black

• 9: Rainbow

• T: Lots of circles

• Shift + Click Flag: Turbo Mobe
